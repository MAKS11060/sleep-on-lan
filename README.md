<h1 align="center">Sleep-on-Lan</h1>
<h3 align="center">Switches the PC to sleep mode using WoL <a href="https://wikipedia.org/wiki/Wake-on-LAN">(Wake-on-LAN)</a></h3>

## Usage

Install [Deno](https://deno.com)

```sh
deno task start
```
