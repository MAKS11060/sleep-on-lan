<h1 style="text-align: center">Sleep-on-Lan</h1>
<h3 style="text-align: center">Switches the PC to sleep mode using WoL (Wake-on-Lan)</h3>

## Usage

Install [Deno](https://deno.com)

```sh
deno task start
```
